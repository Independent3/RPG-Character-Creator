## RPG Character Creator (Python)

A Python program that creates RPG characters with a name and three attributes:
**Strength (STR), Intelligence (INT), and Charisma (CHA)**.  
Each attribute is displayed using a visual dot-based system.

---

### Implemented Validations

- Character name must be a string
- Name cannot be empty or contain spaces
- Name length must be **10 characters or fewer**
- Each stat must be an integer between **1 and 4**
- Total stat points must equal **7**
